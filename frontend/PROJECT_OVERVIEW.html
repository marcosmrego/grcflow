🚀 Como Começar (5 minutos)

1. Configurar Banco de Dados

cd backend cp .env.example .env # Editar .env com credenciais PostgreSQL npm install npm run db:migrate

2. Iniciar Backend

cd backend npm run dev # Servidor rodando em http://localhost:3000

3. Iniciar Frontend

cd frontend python -m http.server 8000 # Ou: npx http-server # Frontend em http://localhost:8000
💡 Dica: Abra 3 terminais - um para o backend, um para o frontend e um para PostgreSQL se necessário.
